<h1 className="font-extrabold xl:text-4xl text-3xl mb-10">Create React App</h1>

**React** adalah sebuah framework User Interface yang dikembangkan oleh **Facebook**. Hal itu membuatnya React
menjadi salah satu web framework yang paling banyak dipakai oleh perusahaan perusaah besar. Dan di
article ini akan menjelaskan konfigurasi awal untuk aplikasi React pertama kamu.

<h2 className="font-bold xl:text-3xl text-2xl my-8">Persiapan</h2>

Kita akan menggunakan Node package manager, jadi kamu harus menginstallnya terlebih dahulu. Untuk mengecek apakah
Node sudah terinstall anda dapat mengetikan perintah :

**node -v**

Jika node sudah terinstall, maka perintah ini akam menggembalikan nomor versi dari Node, seperti

**v12.18.1.**

<h2 className="font-bold xl:text-3xl text-2xl my-8">
  Setting Boilerplate Applikasi
</h2>

Lumayan sulit untuk membuat React app secara manual, tetapi facebbok telah menciptakan sebuah Node package
create-react-app untuk menghandle Boilerplate sebuah react applikasi.

Kita dapat menggunakan npx command untuk menjalankan perintah create-react-app package.

**npm -g install create-react-app**
//
**npx create-react-app myfirstreactapp**
